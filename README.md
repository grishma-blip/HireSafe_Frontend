# HireSafe Frontend

An AI-powered recruitment fraud detection system designed to protect job seekers from sophisticated scams and fraudulent job postings.

## üéØ Project Overview

HireSafe is a modern web application that uses advanced machine learning algorithms to analyze job postings and detect potential recruitment scams in real-time. The platform provides users with instant risk assessment, detailed fraud indicators, and actionable recommendations to help them make informed decisions about job opportunities.
